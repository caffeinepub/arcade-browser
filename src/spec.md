# Specification

## Summary
**Goal:** Add WASD key support to all arcade games alongside existing arrow key controls.

**Planned changes:**
- Update Snake game to accept WASD keys (W/A/S/D for up/left/down/right) in addition to arrow keys
- Update Pong game to accept WASD keys (W/S for up/down paddle movement) in addition to arrow keys
- Update Breakout game to accept WASD keys (A/D for left/right paddle movement) in addition to arrow keys
- Update GameControlsOverlay to display both WASD and arrow key controls for each game

**User-visible outcome:** Players can control all games using either WASD keys or arrow keys interchangeably during gameplay, with both control schemes displayed in the controls overlay.
